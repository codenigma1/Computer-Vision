class ConvAutoEncoder(nn.Module):
    def __init__(self):
        super().__init__()
        # Define encoder
        self.encoder = nn.Sequential(
                            nn.Conv2d(in_channels = 1, out_channels = 32, kernel_size = 3, stride = 3 , padding = 1),
                            nn.ReLU(True),
                            nn.MaxPool2d(2, stride = 2),
                            nn.Conv2d(in_channels = 32, out_channels = 64, kernel_size = 3, stride = 2, padding = 1),
                            nn.ReLU(True),
                            nn.MaxPool2d(2, stride = 1))
        # Define decoder
        self.decoder = nn.Sequential(
                            nn.ConvTranspose2d(in_channels = 64, out_channels = 32, kernel_size = 3, stride = 2),
                            nn.ReLU(True),
                            nn.ConvTranspose2d(in_channels = 32, out_channels = 16, kernel_size = 5, stride = 3, padding = 1),
                            nn.ReLU(True),
                            nn.ConvTranspose2d(in_channels = 16, out_channels = 1, kernel_size = 2, stride = 2, padding = 1),
                            nn.Tanh())
        
        # Define foward method
    def forward(self, x):
        x = self.encoder(x)
        x = self.decoder(x)
        return x

Comprehension of the first convolutional layer: Say, I'm constructing a CNN, and my input layer accepts grayscale images that are 200 by 200 pixels (corresponding to a 3D array with height 200, widht 200, and depth 1). Then, say I'd like the next layer to be a convolutional layer with 32 filers, each filer haveing a widht and height of 3. When performing the convolution, I'd like the filter to jump 3 pixels at a time. I also want the filer to extend outside of the image boundaries; in other words, I want to pad the image with 1.

# Define a function to train on a batch of data (train_batch)
def train_batch(input, model, criterion, optimizer):
    model.train()
    optimizer.zero_grad()
    output = model(input)
    loss = criterion(output, input)
    loss.backward()
    optimizer.step()
    return loss
In [12]:
# Define the function to validate on the batch of data (validate_batch):
@torch.no_grad()
def valid_batch(input, model, criterion):
    model.eval()
    output = model(input)
    loss = criterion(output, input)
    return loss

Grouping similar images using t-SNE¶

"t-SNE" that visualizes high-dimensional data by giving each datapoint a location in a two or three-dimensional map.

For instance:- We can reduce the 64-dimensional vector of convolutional autoencoder to a 2- dimensional space by using a technique called t-SNE.

This way, our understanding that similar images will have similar embeddings can be proved, as similar images should be clustered together in the two-dimensional plane. In the following code, we will represent embeddings of all the test images in a two dimensional plane
